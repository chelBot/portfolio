Space Hopper

Collect all the rubies and try not to get stuck! A simple 2D platformer game created with the Quintus HTML5 framework and the map editor Tiled to demonstrate a basic understanding of game development and object-oriented design. Created in homage to Grace Hopper.

Princess Adventures

Get the key so you can save Hero Boy! If a bug gets you then you become Ghost Girl and must collect all the death gems before the Reaper gets you. This game is not quite finished yet so stay tuned as it's coming soon.

Street Search

A single page app which implements the New York Times and Wikipedia APIs to search for articles related to a street or city of your choice. Demonstrates an understanding of jQuery, promises, and how to use AJAX to interact with server-side APIs. This app is a work in progress that will eventually feature an interactive map of the given location. Stay tuned!

Markov or Trump

Coming Soon! A parody site that uses Markov chains and the BeautifulSoup library to generate random text from Trump's twitter account, speeches, and interviews. How good are you at determining the difference between randomly generated text and the 45th President's rhetoric? The backend of this site is written in python and utilizes the flask microframework.

Code Inspector

A JavaScript inspection API that allows the user to whitelist specific functionality (e.g. "This snippet MUST use a for loop and an if statement"), blacklist specific functionality (e.g. "This snippet MUST NOT use a while loop or a variable declaration"), and to specify the general structure of a code snippet (e.g. "There should be a for loop nested within a while loop"). This program uses Esprima to parse code and to output an Abstract Syntax Tree. All code inspect tests are then preformed on this AST.

Paper Piano

An Arduino project designed to teach children about Nodebots and general STEM topics. A trace is made with conductive ink and graphite (which acts as a pulldown resistor). Paper switches can be thrown to complete the circuit, thereby causing the Arduino to rely a switch-specific frequency to the buzzer. This project uses the Johnny-Five library and was one of several featured in the educational Podpi comic.

Cat Clicker

A single page app strictly for demonstrative purpose. Illustrates an understanding of basic organizational principles namely, using the MVC design pattern to separate concerns. Closures and Immediatedly-Invoked Function Expressions are used to handle some common problems associated with event listeners.
