Space Hopper

Collect all the rubies and try not to get stuck! A simple 2D platformer game created with the Quintus HTML5 framework and the map editor Tiled to demonstrate a basic understanding of game development and object-oriented design. Created in homage to Grace Hopper.

Princess Adventures

Get the key so you can save Hero Boy! If a bug gets you then you become Ghost Girl and must collect all the death gems before the Reaper gets you. This game is a work in progess so stay tuned.

Street Search

A single page app which implements the New York Times and Wikipedia APIs to search for articles related to a street or city of your choice. Demonstrates an understanding of jQuery, promises, and how to use AJAX to interact with server-side APIs. This app is a work in progress that will eventually feature an interactive map of the given location. Stay tuned!

Cat Clicker

A single page app strictly for demonstrative purpose. Illustrates an understanding of basic organizational principles namely, using the MVC design pattern to separate concerns. Closures and Immediatedly-Invoked Function Expressions are used to handle some common problems associated with event listeners.
